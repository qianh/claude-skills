# 试题模板参考

本文档提供各学科常见题型的标准模板和格式示例。

## 一、理科试题模板

### 数学

#### 选择题模板

```

```

```

```

## 四、答案格式规范

### 选择题答案格式

```
一、选择题（每题3分，共30分）

1. C
   解析：根据题意，只有选项C符合定义。A选项错误在于...，B选项...

2. B
   解析：...
```

### 填空题答案格式

```
二、填空题（每题4分，共20分）

1. {2, 3}
   要点：交集是两个集合共有的元素

2. Na₂O₂（过氧化钠）
   要点：注意是燃烧而非氧化，生成物不同
```

### 计算题答案格式

```
三、计算题（共30分）

1. （10分）
   解：(1) f(x) = x² - 2x + 1 = (x-1)² ≥ 0  ......... 2分
          当 x = 1 时，f(x)min = 0  ......... 2分
          答：最小值为0。 ......... 1分

      (2) f(0) = 1, f(3) = 4  ......... 2分
          因为 f(x) 在 [1, +∞) 上单调递增  ......... 1分
          所以 f(x)max = f(3) = 4  ......... 1分
          答：最大值为4。 ......... 1分

   评分标准：
   - 配方正确得2分
   - 求出最小值得2分
   - 答题格式规范得1分
   - 求出端点函数值得2分
   - 判断单调性得1分
   - 求出最大值得1分
   - 答题格式规范得1分
```

### 简答题答案格式

```
四、简答题（共20分）

1. （8分）
   参考答案：
   (1) 力是物体对物体的作用；（2分）
   (2) 力不能离开物体而独立存在；（2分）
   (3) 力的作用是相互的；（2分）
   (4) 力的作用效果有两个：改变物体的运动状态和改变物体的形状。（2分）

   评分标准：
   - 答出任意3点得6分，每点2分
   - 答出全部4点得8分
   - 表述不准确，酌情扣1-2分
```

## 注意事项

1. **分值标注**：每道题目都应明确标注分值
2. **图片质量**：图例题的图片必须清晰、专业
3. **答案详细性**：答案不仅要给出结果，还要有解析和评分标准
4. **格式统一**：同类题型格式应保持一致
5. **难度梯度**：试题应有明显的难度层次
6. **时间分配**：题量应与考试时间相匹配
